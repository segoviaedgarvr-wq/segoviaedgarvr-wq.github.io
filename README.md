# Asociación de Agua Potable La Laguneta 1 y 2

Este es un proyecto de **desarrollo de software en código abierto** diseñado para servir a la comunidad de La Laguneta. La web proporciona información esencial sobre el servicio de agua potable, avisos y contacto.

## 📋 Objetivos del Proyecto
* Proporcionar una presencia digital a la Asociación.
* Facilitar la comunicación de cortes o mantenimientos del servicio.
* Cumplir con los requisitos de la tarea de desarrollo open source.

## 🚀 Tecnologías Usadas
* **HTML5 & CSS3:** Estructura y diseño estático.
* **GitHub Pages:** Hosting gratuito y despliegue continuo.

## 💻 Contribuir
Al ser un proyecto de código abierto, aceptamos contribuciones. Si encuentras un error o quieres mejorar el diseño, por favor abre un "Issue" o envía un "Pull Request".
